# JHDynamicFont
通过为UIFontDescriptor添加category方法实现动态改变指定控件（如UILabel UIButtonLabel等）的字体大小。

## 详细使用方法
[简书链接](http://www.jianshu.com/p/a1339d45a324)

## 已上传至cocoapods trunk

可以直接在终端使用下面命令来搜索

`
pod search JHDynamicFont
`
